Use full Windows commands 

In Day to day regular system admin activites i have used below windows commads. I hope it will be usefull some on working on Windows Administration.

  • Open network connections: Runàncpa.cpl
  • Open firewall: Runàfirewall.cpl
  • Open services console: Runàservices.msc
  • Open system properties: Runàsysdm.cpl
  • Open Disk management: Runàdiskmgmt.msc
  • Open Chrome browser: Runàchrome
  • Open MS Word: RunàWinword
  • Open MS outlook: Runàoutlook
  • Batch command to restart: shutdown /r /t 01
  • Open control panel: Runàservices.msc
  • Open Remote desktop connections: Runàmstsc
  • Open Task manager: Runàtaskmgr
  • Open program and Features: Runàappwiz.cpl
  • Open C:\Windows\System32\drivers: Runàdrivers
  • Open MS excel: Runàexcel
  • Open MS paint: Runàmspaint
  • Checking system IP Address: ipconfig /all
  • Renew IP address: ipconfig /renew
  • Flush DNS cache: Ipconfig /flushdns
  • Running check for update: wuauclt.exe /updatenow
  • Open group policy edit: Runàgpedit.msc
  • Open registry: Runàregedit
  • Open Windows explorer: Runà explorer
  • Open Notepad: RunàNotepad
