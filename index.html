Use full Windows commands 

In Day to day regular system admin activates I have used below windows commands. I hope it will be useful some on working on Windows Administration.

  • Open network connections: Run type ncpa.cpl

       

     

  • Open firewall: Run type firewall.cpl

       

     

   

  • Open services console: Run type services.msc

     

     

   

  • Open system properties: Run type sysdm.cpl

     

     

  • Open Disk management: Run type dismgmt.msc

     

     

  • Open Chrome browser: Run type chrome

     

  • Open MS Word: Run type winword

     

  • Open MS outlook: Run type outlook

     

  • Batch command to restart: shutdown /r /t 01
  • Open control panel: Run type control

     

  • Open Remote desktop connections: Run type mstsc

     

  • Open Task manager: Run type taskmgr

     

  • Open program and Features: Run type appwiz.cpl

     

     

  • Open C:\Windows\System32\drivers: Run type drivers

         

  • Open MS excel: Run type excel

     

  • Open MS paint: Run type mspaint

     

     

  • Checking system IP Address: open cmd and type ipconfig /all
  • Renew IP address: ipconfig /renew
  • Flush DNS cache: Ipconfig /flushdns
  • Running check for update: wuauclt.exe /updatenow
  • Open group policy edit: open run type gpedit.msc
  • Open registry: open run regedit.msc
  • Open Windows explorer: open run type explorer
  • Open Notepad: open Run type notepad
